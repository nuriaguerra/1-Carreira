## De que vai esta materia

Programación II continúa onde deixou Programación I, pero dá un salto importante: aquí empezan as **estruturas de datos**. É o momento en que o código deixa de ser unha secuencia de instrucións e empeza a organizarse de forma máis sofisticada. Para moita xente, esta materia é bastante máis sinxela que Progra I.
