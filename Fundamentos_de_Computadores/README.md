# Fundamentos de Computadores

> Hardware · Primeiro curso

---

## De que vai esta materia

Fundamentos de Computadores responde á pregunta que moita xente se fai pero non sabe como formular: *¿como funciona realmente un ordenador?* Non o software, non o sistema operativo — o propio hardware. Como a CPU executa instrucións, como a memoria garda datos, como se comunican os compoñentes.

É a materia que che permite pasar de "o ordenador fai cousas" a "o ordenador fai cousas **así**". Ese cambio de perspectiva é moi valioso.

---

## Contidos habituais

- Organización xeral dun computador: CPU, memoria, E/S
- Representación da información: enteiros, coma flotante (IEEE 754), caracteres
- Conxunto de instrucións (ISA): tipos de instrucións, modos de enderezo
- Ciclo de instrución: fetch, decode, execute
- Introdución á arquitectura RISC vs CISC
- Ensamblador básico (MIPS, x86 ou ARM segundo o temario)
- Xerarquía de memoria: rexistros, caché, RAM, disco
- Introdución ao pipeline
- Entrada/saída e interrupciones

## Consellos

- A representación en coma flotante IEEE 754 é complicada e aparece en exame. Non a deixes para o final.
- O ensamblador parece imposible as primeiras horas. Logo de escribir uns poucos programas pequenos, todo encaixa. Aguanta.
- Entender a xerarquía de memoria explica por que o código ben escrito é máis rápido que o mal escrito. Tes un motivo moi concreto para aprendela.
- Relaciona esta materia con Sistemas Dixitais: a CPU está feita dos circuítos que estudaches alí.
