# Fundamentos de Computadores

> Hardware · Primeiro curso

---

## De que vai esta materia

Fundamentos de Computadores responde á pregunta que moita xente se fai pero non sabe como formular: *¿como funciona realmente un ordenador?*. Como a CPU executa instrucións, como a memoria garda datos, como se comunican os compoñentes.

As prácticas son algo complexas porque tes que aprender unha nova linguaxe, **ensamblador**.
