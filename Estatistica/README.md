> Matemáticas · Primeiro curso

## De que vai esta materia

Estatística dáche ferramentas para describir, analizar e sacar conclusións a partir de datos. É fundamental en intelixencia artificial, ciencia de datos, validación de algoritmos e case calquera área onde haxa que tomar decisións con incerteza.

A parte de probabilidade pode resultar contraintuitiva ao comezo — é normal. Con práctica encaixa.
