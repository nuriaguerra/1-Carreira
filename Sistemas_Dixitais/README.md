# Sistemas Dixitais

> Hardware · Primeiro curso

---

## De que vai esta materia

Sistemas Dixitais explica como se constrúen os circuítos que procesan información binaria — desde as portas lóxicas máis simples ata circuítos secuenciais capaces de gardar estado. É a materia que che mostra como o hardware pensa en 0s e 1s.
Natalia é a mellor profe para dar as prácticas.
