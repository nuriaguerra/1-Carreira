## De que vai esta materia

Esta materia é a base lóxica de todo o resto. Aquí aprendes a pensar e escribir matematicamente de forma rigorosa: que é unha demostración, como funciona a lóxica formal, e as estruturas fundamentais sobre as que se constrúe toda a matemática.

Para moita xente é o primeiro contacto real coas matemáticas como linguaxe, non como ferramenta de cálculo. O salto pode custar, pero vale moito a pena.
